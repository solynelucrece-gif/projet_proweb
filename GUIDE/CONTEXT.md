# PROJET DE PROGRAMMATION WEB AVEC PYTHON

# Présentation

Le présent projet vous donne libre accès à votre imagination pour produire une
application web qui présente des indicateurs permettant de mettre en valeur les données
présentes dans le fichier data.xlsx joint. Le projet doit présenter à minima 4 pages web
parmi lesquelles une page d’accueil qui présentera le projet et les membres de l’équipe.
Cette page doit être accessible au public sans authentification.

Le fichier data.xlsx contient un échantillon de 50000 lignes de données recueillies sur des
comptes Instagram. Vous devez l’analyser et en tirer des indicateurs pertinents à
présenter dans votre application. Si besoin, le fichier data pourra être transformé en
fichier csv pour son exploitation.

# Consigne

- Le projet doit être conçu avec la version 5.2.7 de Django
- Un environnement virtuel (devoir_env) doit être créé et intégré au dossier du projet.
- Un fichier requirements.txt contenant toutes les exigences du projet doit être créé.
- Concevoir la base de données sous postgreSQL ou MySQL
- Vous pourrez utiliser toutes les librairies python et autres que vous trouverez
    nécessaire.
- Toutes les pages de l’application doivent être accessible après authentification à
    l’exception de la page d’accueil et de la page de contact.
- La page d’accueil doit présenter le projet ainsi que les membres de l’équipe.
- Une page contact doit permettre d’envoyer un mail sur une adresse électronique
    qui sera paramétrée dans le settings.py
- Une page de profil doit être créée et présenter les informations de l’utilisateur
    connecté. Elle doit permettre à l’utilisateur de modifier ses informations
    personnelles. Un utilisateur doit avoir sur son profil les informations suivantes à
    minima : Nom, Prénom, Sexe, email, Fonction, Photo de profil.
- La gestion des accès doit être paramétrée de sorte que certaines informations ne
    soient pas accessibles à tous les utilisateurs.
- La page de l’administrateur principal doit être paramétrée et accessible à partir de
    l’authentifiant suivant _:_ **_Nom utilisateur_** _: admin_ et **_mot de passe_** _: AS3admin_

# Livrables

- Le dossier du projet contenant l’environnement virtuel
- Le fichier requirements.txt
- L’export de la base de données avec les données enregistrées.
- Un fichier pdf contenant le manuel d’utilisation de l’application. Il doit contenir la
    liste des utilisateurs de l’application ainsi que leurs mots de passe, les
    informations qu’ils sont autorisés à voir, et toute information utile pour l’utilisation
    de l’application. Il doit présenter les indicateurs sélectionnés.
